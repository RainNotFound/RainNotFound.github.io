    知识扫盲

    新建文件

    新建文件的拓展名必须以 “$.c$” 为结尾

    编译

    还没学过《编译原理》这门课啦，就只能会操作不懂原理。

    在cmd命令框中先用 “cd+c文件存放路径“ 到达你存放c文件的文件夹，然后输入 “gcc 文件名.c”

    若语法没有错误会在当前文件夹生成一个 “a.exe” 的可执行文件，双击或者在cmd中输入 “a.exe” 就会运行。若有语法错误则会告诉你哪里有错误。

    默认编译的名称为 “a.exe” ,也可以输入 “gcc 文件名.c -o 你想的名字” 来改变编译后文件生成名字。

    C程序的基础框架

    #include <stdio.h>	//编译预处理指令

int main()			//程序的主入口函数main
{					//程序（函数、功能）开始的标志
    
    
    return 0;		//程序退出前返回给调用者（操作系统）的值
}					//程序（函数、功能）结束的标志

    “//“是注释，本行后面的内容不会被编译

    “/* */“ 星号之间的内容不会被编译

    变量

    声明变量

    顾名思义可变的量，像数学表达式的$x,y,z$ 啊，要先声明再使用

    怎么声明呢，像下面那样

    数据类型+变量名
int a = 3;
int b;
b = a+1;

    变量有四个属性：数据类型、变量名、变量值、地址（储存单元）

    按我目前理解，数据类型是申请多大的空间，存储单元是申请的地址。

    变量名称

    变量的名只能由 “字母、数字、下划线” 组成，且不能以数字开头

    c中是会区分大小写的

    命名方法常采用 “驼峰命名法” 来增加同行之间的交流
